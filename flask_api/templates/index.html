Using CURL to test the ImageQ API

The ImageQ API currently relies on the 1000 classes provided by the imagenet weights available for use on keras.
Below you can see the image we wish to classify, a dog, but more specifically a beagle:

        
        $ curl -X POST -F image=@dog.jpg 'http://imageqapi.appspot.com/predict'
